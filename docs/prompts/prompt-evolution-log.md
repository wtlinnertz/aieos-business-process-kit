# Prompt Evolution Log — Business Process Kit

This log tracks significant changes to BPK prompts and the reasoning behind them.

---

## Initial Release (v1.0)

**Date:** 2026-03-13

All three prompts (PIA, TP, RC) created as part of the initial BPK build.

**Design decisions:**
- PIA prompt emphasizes traceability to SAD/TDD — prevents speculative process impacts
- TP prompt enforces measurable training criteria — "attended training" is explicitly rejected
- RC prompt positioned as evidence collection, not generation — the AI organizes evidence, it doesn't create it
